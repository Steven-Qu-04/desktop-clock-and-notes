# 时钟+便签简约桌面

欢迎使用简约桌面，下载点击time.html即可开始使用。

## 时间查看功能

桌面就可以看时间哈哈哈哈，F11全屏效果最佳。

## 便签功能

双击即可设置便签，所有便签可拖动，默认拖动便签在最上层，便签右上角设置隐藏、删除按钮。便签存储位置为浏览器缓存，推出、关机可保留，但注意不要切换打开html的浏览器，否则在新浏览器的缓存中无法找到之前存储的便签。

## 便签时间轴

单击右侧小按钮即可打开侧边栏。

- 便签管理：可以在这里设置便签可见性或删除。
- 提醒设置：在便签下方时间栏选择一个时间，到时间便签即可自动显示、置顶并呼吸高亮，单击便签即可解除高亮状态。
- 一键导出：右上角提供“日历导出”（导出为ICS文件）、txt导出选项。

# 背景图片

默认背景图片由gemini生成，可自行更改背景图片，更改方式为在 `time.html`搜索 `.png`找到图片位置，更改路径引用即可。
